这一篇文章我们讲 OMV 上安装扩展插件。

<!-- more -->

> 本文适用于 OMV5。

接上一篇教程，咱们装完了OMV系统，但是刚刚装完的系统，插件会特别少，我们需要安装扩展才可以。

## 安装

使用root用户登录到omv。

```
ssh root@omv
```

然后安装插件

```
wget -O - https://github.com/OpenMediaVault-Plugin-Developers/packages/raw/master/install | bash
```

安装之后就可以发现，插件变多了。
